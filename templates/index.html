API Reference ✕
GET /api/streams
Retrieve all configured streams. Returns list of {id, url, name}.
POST /api/streams
Add a new stream. Body: {"url": "...", "name": "..."}. Returns ID.
PUT /api/streams/<id>
Update stream. Body: {"url": "...", "name": "..."}.
DEL /api/streams/<id>
Delete a stream by ID.
GET /video_feed/<id>
MJPEG video stream for live playback. Browser compatible.
GET /api/snapshot/<id>
Get single JPEG snapshot (cached).
